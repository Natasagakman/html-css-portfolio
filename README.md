# Персональне портфоліо — Наталія Гакман

Цей проект є особистим веб-сайтом, створеним для демонстрації навичок веб-розробки та представлення професійної інформації.

## Про автора
Я студентка Чернівецького національного університету імені Юрія Федьковича, навчаюся за спеціальністю «Електронні комунікації та радіотехніка». Окрім навчання, цікавлюся фотографією пейзажів та дизайном.

## Технології та інструменти
При розробці проекту використовувалися наступні інструменти:
- Мови розмітки та стилів: HTML5, CSS3.
- Середовище розробки: Visual Studio Code.
- Система контролю версій: Git.

## Функціонал та розділи
На сайті реалізовано кілька основних блоків:
- Головна сторінка з короткою біографією.
- Розділ про захоплення (читання книг, прогулянки та фотографія).
- Контактна інформація та навігація.

## Робота з проектом
Репозиторій використовується для ініціалізації лабораторних робіт та відстеження змін у коді. Для локального перегляду достатньо відкрити файл index.html у будь-якому сучасному браузері.
